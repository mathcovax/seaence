### start container
```bash
npm run docker:abys
```

### cook document
```bash
npm run docker:abys:cookNodeSameRawDocument
```

### export baked document to sea
```bash
npm run docker:abys:exportBakedDocument
```

### run fixtures
```bash
npm run docker:abys:fixtures
```
**Options:**
- `-b`, `--numberOfBakedDocuments` 100 # number
